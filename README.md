# style-linux-terminal

* Update Linux box
```
sudo apt update
```
```
sudo apt upgrade
```
> _**Note**: If you already have installed Git and cURL, you can skip next 2 steps._
* Installing cURL
```
sudo apt install curl
```
* Installing Git
```
sudo apt install git-all
```
